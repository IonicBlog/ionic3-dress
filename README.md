# 关于项目

此项目是基于ionic3+angular4开发的一款导购APP，主要作为教程源码给大家共享。

# 如何使用

## 安装CLI

```bash
$ npm install -g ionic cordova
```

## 安装node_modules

```bash
$ npm install
```

## 浏览器调试

```bash
$ ionic serve
```
